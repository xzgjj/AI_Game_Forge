# 测试与构建命令流程（推荐）

## 1. 进入项目目录

```powershell
cd D:\app_project\AI_Game_Forge
```

## 2. 一键工作流命令（推荐）

```powershell
# 代码检查
npm.cmd run wf:check

# 前端测试 + Rust 测试
npm.cmd run wf:test

# 前端构建
npm.cmd run wf:build

# 检查 + 测试 + 构建（完整流程）
npm.cmd run wf:all

# 清理构建产物（dist/build/.svelte-kit/target/src-tauri/target）
npm.cmd run wf:clean

# 深度清理（额外删除 node_modules）
npm.cmd run wf:clean:deep

# Tauri 本地开发运行
npm.cmd run wf:tauri:dev

# Tauri 打包构建
npm.cmd run wf:tauri:build
```

## 3. 常用场景

```powershell
# 日常开发前
npm.cmd run wf:check
npm.cmd run wf:test

# 发版前
npm.cmd run wf:all
npm.cmd run wf:tauri:build

# 磁盘空间清理
npm.cmd run wf:clean
```

## 4. 备注

- `scripts/workflow.ps1` 会自动检查 Node/npm/cargo，并在当前会话补充 cargo 路径。
- `wf:clean` 不会删除受 Git 跟踪的 `src-tauri/gen/schemas` 文件，避免仓库误变脏。
