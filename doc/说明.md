

# java + ai

通过AI辅助降低游戏开发门槛，让创意者无需编程即可快速原型化游戏创意，实现"创意即游戏"的无缝转化。

dist/ 目录 - 构建输出目录                                                       
  作用：存放前端代码的构建产物（编译、打包后的最终文件）。                        
  
  内容：
  - index.html - 入口HTML文件
  - assets/ - 打包后的JavaScript、CSS、图片等静态资源
  - 其他编译后的前端文件

  生成方式：运行 npm run build 或 vite build
  命令时，Vite构建工具会将源代码编译、打包、优化后输出到此目录。

  在Tauri中的作用：Tauri桌面应用运行时，会加载dist/目录中的前端文件作为应用的GUI界
  面。


----------------------------------------------
  node_modules/ 目录 - Node.js依赖库

  作用：存放项目所有npm依赖包（包括运行时依赖和开发依赖）。

  内容：
  - @tauri-apps/ - Tauri相关API和插件
  - svelte/ - Svelte框架核心
  - typescript/ - TypeScript编译器
  - vite/ - 构建工具
  - eslint/, prettier/ - 代码质量工具
  - 其他依赖包（约200MB）

  生成方式：运行 npm install 时，根据package.json中的依赖声明自动下载安装。       

  特点：
  - 体积庞大，不应提交到Git仓库
  - 可以在任何机器上通过npm install重新生成
  - 包含平台特定的二进制文件

json - 依赖版本锁文件
